# Pulse Terminal ⚡

A modern, minimal, dark GUI terminal built from scratch in Python using PySide6.

<img src="assets/img.png" width="500"/>

## Features
- Dark sleek UI with green prompt
- Run any shell command (ls, pwd, grep, etc.)
- Built-in `cd` with full path resolution
- Live prompt updates as you navigate directories

## Tech Stack
- Python 3
- PySide6 (Qt6)

## Getting Started

```bash
pip install PySide6
python src/main.py
```
